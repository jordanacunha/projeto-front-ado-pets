# Introdução
O Aumigo é um site desenvolvido para desktop com o objetivo de facilitar a adoção de animais, conectando pessoas a pets que aguardam um novo lar. A plataforma oferece navegação simples e direta, permitindo que o usuário visualize os animais disponíveis e conheça suas histórias de forma clara e acessível.

# Tela inicial
Mostra um banner com vários animais e um botão “Quero adotar!”. Abaixo, apresenta duas pequenas seções explicando como funciona o site e incentivando a adoção.

# Lista de pets
Exibe os animais disponíveis em uma grade, com foto, nome e localização. O usuário pode visualizar vários pets ao mesmo tempo e clicar em um deles para ver mais detalhes.

# Detalhes dos pets
Mostra a foto grande do animal, suas informações principais (nome, idade, espécie, porte, localização) e um texto curto com sua história. Inclui o botão “Quero adotar!” em destaque.
